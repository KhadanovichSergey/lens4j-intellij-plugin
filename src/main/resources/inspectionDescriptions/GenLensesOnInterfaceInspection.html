This inspection reports when the @GenLenses annotation is used on interfaces.
@GenLenses annotation can be used only with top-level classes. 


    // incorrect
    @GenLenses(lenses = @Lens(path = "name"))
    interface Fly {
        String getName();
    }



    // correct
    @GenLenses(lenses = @Lens(path = "name"))
    class Bird {
        String name;
    }
