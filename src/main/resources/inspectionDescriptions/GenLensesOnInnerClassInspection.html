This inspection reports when the @GenLenses annotation is used on inner classes.
@GenLenses annotation can be used only with top-level classes. 


    // incorrect
    class Outer {
        @GenLenses(lenses = @Lens(path = "name"))
        class Inner {
            String name;

            // accessors
        }
    }



    // correct
    @GenLenses(lenses = @Lens(path = "name"))
    class Outer {
        String name;

        // accessors
    }
